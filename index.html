CSS Animations

CSS Animations are a great way for you to add spark and design to your website. If you have a creative edge, you can convey that through CSS Animations. To animate in CSS, we must have a proper understanding of the CSS property transition. Transitions are a set of properties that allow for smooth animation and movement in between transforms. The CSS transition properties available to us are the most important are transition-property, -duration, timing-function, and -delay. Transition-property verifies the CSS property which will be affected by this transition. This could be a button with a :hover feature that you decide to make, and if you apply a back ground-color, this will ensure the button with the rule of the transition set on it. Transition-duration decide s how long it will take for this transition to occur, transition-timing-function will set the acceleration cur ve for the animation, (e.g. ease, ease-in, ease-out), and transition-delay will model how long it will take to start. Transform is how change the state of an element in order to give off the illusion of animation. We can use translateX/Y/Z, which will move the element to the right, the left, or forward and backwards in 3d space, or we can use rotate. This function will rotate the element based on a angular unit (deg or rad)
